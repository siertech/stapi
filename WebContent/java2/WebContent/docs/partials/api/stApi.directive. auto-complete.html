auto-complete
directive in module stapi

Usage

as attribute
<ANY auto-complete
     label="{String@}"
     object-name="{String@}"
     attr="{String@}"
     [ng-model="{Object}"]
     initial-busca="{String@}"
     extra-class="{String@}"
     label-cad="{String@}"
     place-holder="{String@}"
     [value-only="{Boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
labelString@

Texto que aparecerá acima da caixa de texto, ex: Nome do cliente

objectNameString@

Nome do objeto que será utilizado na pesquisa (Em camel case), ex: Cliente

attrString@

Atributo do objeto que será utilizado na pesquisa , ex: nome

ng-model
(optional)
Object

Objeto no escopo a qual o elemento está relacionado , ex: empresa

initialBuscaString@

Expressão padrão para busca de objeto, ex: Paulo

extraClassString@

Classe css extra a ser adicionada ao elemento input da diretiva

labelCadString@

Texto a ser exibido no botão de cadastro de novo item

placeHolderString@

place-holder do elemento

valueOnly
(optional)
Boolean

se true, não seta o objeto inteiro no model, apenas o valor(objjeto[attr])

Example

Exemplo utilizando Um Objeto

<auto-complete 
	        label="Nome do cliente"
	        object-name="Cliente"
	        attr="nome",
	        ng-model="empresa" 
            initial-busca="João"  
		    extra-class="input-lg"  
		    label-cad="Cadastrar novo Cliente" 
		    place-holder="Primeiro, digite o nome do cliente"  
		    >
		 </auto-complete>

Exemplo utilizando com.siertech.stapi.opcao

<auto-complete     
		    label-cad="Cadastrar categoria teste" 
		    place-holder="placeHolderCategoria"  
		    object-name="Opcao"  
		    attr="valor" label="Descrição"
		    fix-properties="{descricao:'categoria_teste'}" 
		    ng-model="categoria" 
		    value-only="true"
		    initial-busca=""
		  </auto-complete>
